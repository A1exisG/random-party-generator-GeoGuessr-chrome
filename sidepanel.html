Generator

With this generator, you can randomly generate a map and game parameters.

Step 1: Select the information you wish to randomly generate

Map : Randomly generate the map
Rounds : Randomly generate the number of rounds
Time : Randomly generate the duration of a round
Move : Randomly generate whether you can move
Pan : Randomly generate if you can move the camera
Zoom : Randomly generate if you can zoom

Step 2: Click on the “Random Game” button

If you have generated a random map, you will be automatically redirected to the map page with all parameters applied.

If you haven't generated a map, then the page you're on will simply refresh and the settings will be applied.
